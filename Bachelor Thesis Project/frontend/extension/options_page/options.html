About Hear Me Assistant

What is Hear Me Assistant?

Hear Me Assistant (HMA) is a Google Chrome extension that aids visually impaired people in website accessibility by getting an image description in case one is missing or providing an audio description of the feed posts of some well known social media websites: Facebook, Instagram and Linkedin. The extension functionalities can be configured by going to settings section and selected one of the preferred mode: Image Description Mode, Social Media Mode or both of them.

What functionalities does the application have?

Hear Me Assistant can do the following:

Image Description Social Media Feed
Generate a caption for an image Provide a post description which includes the author, the post text and the image caption for the feed posts
Add the generated description for each image on the website The user is able to listen to each post description
The screen reader can now also describe images by using the generated caption Provide more posts description on page scrool

General Options

Image Description Mode

The mode which provides a caption for each image of the active website page such that the screen reader of the user is able to describe the image.

Social Media Mode

This mode permits the user to automatically get audio description of each feed post when login to one of the following social media platforms: Facebook, Instagram or Linkedin. The description includes a short caption of the image of the post(if it exists), the description text and the author of the post.
